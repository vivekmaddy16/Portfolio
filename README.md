# Vivek Maddheshiya Portfolio

A responsive single-page developer portfolio built with HTML, CSS, and vanilla JavaScript. This project presents Vivek Maddheshiya's profile, education, featured projects, certifications, and contact details in a polished personal website format.

## Repository

GitHub: `https://github.com/vivekmaddy16/Portfolio`

## Overview

This portfolio is designed as a static front-end website with a strong visual presentation and smooth interactions. It uses a custom layout, animated sections, and a Three.js-powered hero background to create a modern personal brand site without any build step or framework setup.

## Features

- Responsive single-page portfolio layout
- Animated hero section using `Three.js`
- Scroll reveal effects and smooth anchor navigation
- Interactive project preview popup
- Custom cursor and hover interactions
- Dedicated sections for about, education, projects, certifications, and contact

## Tech Stack

- `HTML5`
- `CSS3`
- `JavaScript`
- `Three.js` loaded via CDN
- Google Fonts

## Run Locally

No installation is required.

1. Clone the repository:

```bash
git clone https://github.com/vivekmaddy16/Portfolio.git
```

2. Open the project folder.
3. Open `index.html` directly in your browser.

For a smoother development experience, you can also run it with a local static server such as VS Code Live Server.

## Customization

- Update personal details, section content, and links in `index.html`
- Modify animations and interactions in `script.js`
- Adjust colors, spacing, typography, and responsive styles in `style.css`
- Replace images and resume files inside the `assets/` folder
